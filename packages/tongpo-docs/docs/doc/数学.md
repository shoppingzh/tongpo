## 获取范围中的可用子范围

```js
getUnuseRagnes(min, max, usedRanges = [])
```

**参数**

1. min(Number): 范围最小值
2. max(Number): 范围最大值
3. usedRanges(Array\<{ from: Number, to: Number }\>): 已用范围集合

**返回值**

(Array\<{ from: Number, to: Number }\>): 可用范围集合

**例子**

```js
getUnuseRagnes(1, 10, [
  { from: 2, to: 3 },
  { from: 7: to: 8 }
])
// =>
/**
 * [
 *    { from: 1, to: 1 },
 *    { from: 4, to: 6 },
 *    { from: 9, to: 10 }
 * ]
 * /
```

## 获取范围内的随机值

```js
getRandomInRange(min, max)
```

**参数**

1. min(Number): 最小值
2. max(Number): 最大值

**返回值**

(Number): 随机数

**例子**

```js
getRandomInRange(2, 3)
// => 2.1483673
```
