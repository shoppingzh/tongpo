## 合并对象

合并目标对象中值为undefined或null的键。

```js
mergeWithNil(object, sources)
```

**参数**

1. object(Object): 被合并的目标对象
2. sources(Array\<Object\>): 合并源对象

**返回值**

(Object): 合并后的目标对象

**例子**

```js
const target = {
  name: null,
  addr: {
    province: 'Guangdong',
    city: undefined
  }
}
const source = {
  name: 'xpzheng',
  addr: {
    province: 'Guangdong',
    city: 'Shenzhen'
  }
}
mergeWithNil(target, source)
// =>
/**
 * {
 *    name: 'xpzheng',
 *    addr: {
 *      province: 'Guangdong',
 *      city: 'Shenzhen'
 *    }
 * }
 * /
```
