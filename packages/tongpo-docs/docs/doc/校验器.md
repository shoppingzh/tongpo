## 是否为外部链接

```js
isExternalLink(url)
```

**参数**

1. url(String): 链接

**返回值**

(Boolean): 判断结果

**例子**

```js
isExternalLink('http://www.google.com')
// => true
isExternalLink('/users')
// => false
```

## 是否非空字符串

```js
isNotBlank(str)
```

**参数**

1. str(String): 字符串

**返回值**

(Boolean): 校验结果

**例子**

```js
isNotBlank('')
// => false
isNotBlank('  ')
// => false
isNotBlank('123')
// => true
isNotBlank(null)
// => false
isNotBlank(undefined)
// => false
```

## 是否为空字符串

```js
isBlank(str)
```

**参数**

1. str(String): 字符串

**返回值**

(Boolean): 校验结果

**例子**

```js
isBlank(null)
// => true
isBlank(undefined)
// => true
isBlank('')
// => true
isBlank('   ')
// => true
isBlank('123')
// => false
```
